# Terminal Chatbot

A conversational AI chatbot that runs in the terminal, built with Python and the Groq API.

## Features
- Persistent conversation memory across turns
- Customizable system prompt for personality/instructions
- Powered by Llama 3.1 via Groq API

## Setup
1. Clone the repo
2. Install dependencies: `pip install groq python-dotenv`
3. Create a `.env` file with your Groq API key:
4. Run: `python python_.py`

## What I learned
- Making API calls to LLM providers
- Managing conversation history manually (stateless API)
- System prompts and prompt engineering
- Secure API key handling with environment variables
